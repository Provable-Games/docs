# SURVIVOR Token

Information about the SURVIVOR token

The SURVIVOR token is the native governance and utility token for Loot Survivor, designed to decentralize ownership and incentivize participation in the Loot Survivor ecosystem.

## Overview

The SURVIVOR token serves as the cornerstone of the Loot Survivor ecosystem, providing governance rights, in-game utility, and value accrual mechanisms for participants. With a total supply of 10 million tokens, the distribution is carefully structured to reward the community while ensuring sustainable long-term development.

## Key Features

### Decentralized Governance
SURVIVOR token holders control the governance treasury, which includes proceeds from the DCA token sale and the treasury reserve allocation. Token holders determine how these funds are utilized for ecosystem development, partnerships, and community initiatives.


### Value Accrual
The token captures value through the governance treasury, which receives:
- Proceeds from the DCA token sale
- 80% of proceeds from dungeon ticket purchases
- The 500,000 treasury reserve tokens

These funds can be directed towards buybacks, player rewards, and ecosystem development as determined by SURVIVOR token holders.

### Community First
75% of the total supply is allocated to the community through various distribution mechanisms, ensuring broad ownership and aligned incentives.

## Security

The SURVIVOR token contracts are built using the latest stable OpenZeppelin components, which have undergone extensive security audits and are battle-tested across thousands of projects. This foundation ensures the reliability and security of the governance system through proven, industry-standard implementations.

## Disclaimer

The SURVIVOR token is designed to decentralize ownership and governance of the Loot Survivor protocol. It represents participation rights in the ecosystem and should not be considered as a financial investment.

Participation in Loot Survivor and holding SURVIVOR tokens involves risks including smart contract vulnerabilities, market volatility, and game balance changes. Users should understand these risks before participating.
